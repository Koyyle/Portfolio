Linear Rift
How far can you go? A time travel puzzle game inspired by games like Portal and Titanfall. Currently in active (but slow) development. Demo available.
Project Details / Background
Started development in 2019 on roblox, with a demo being released in 2020. Currently being reimagined in the Unity game engine, solely by me.
I was originally a designer and writer on this project, and handled some marketing material. Now, I solely develop the game, and have been making improvements to it's story and mechanics. Currently only the demo is available, but the new demo should be ready by Q3 2024.
